# Setup Instructions

## Prerequisites
- Ensure you have [Node.js](https://nodejs.org/) installed.
- Install [npm](https://www.npmjs.com/) if not already included.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/mqdo/lixi.git
   cd lixi
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```

## Running the Application
To start the application, run:
```bash
npm start
```
